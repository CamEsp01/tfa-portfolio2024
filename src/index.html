Skills

What I like about HTML is that it allows you to easily create the structure of web pages. It is a language that is simple to understand and use. I enjoy organizing content to make websites clear and pleasant to use.

What I love about CSS is that it allows me to transform ideas into beautiful and interactive designs. With CSS, I can create custom styles that adapt to all screens and add animations to make websites more dynamic. I am proficient in techniques like flexbox, CSS grid, and tools like SASS, which enable me to create efficient and easy-to-maintain styles.

I look forward to continuing to learn and develop more complex projects as I progress.

I'm new to the world of PHP, but I'm really excited about what I've discovered so far, just like with JavaScript. I love how this language allows me to create dynamic websites. Although I'm still learning, I'm motivated to improve. I'm open to all opportunities to practice and progress in PHP.
